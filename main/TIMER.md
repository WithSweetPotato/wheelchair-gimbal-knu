## < Arduino Mega 2560 하드웨어 타이머 >

---

| Timer | Usage |
|:---|:---|
| Timer 0 (8bit) | millis(), micros(), delay() |
| Timer 1 (16bit) | ShiftRegPWM 출력 |
| Timer 2 (8bit) | tone() | 
| Timer 3 (16bit) | - |
| Timer 4 (16bit) | 적외선 통신 (IRremote-custom) |
| Timer 5 (16bit) | 서보 모터 제어 (Servo-avr-custom)  |
